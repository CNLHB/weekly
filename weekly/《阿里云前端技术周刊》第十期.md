- JavaScript 语言采用的是单线程模型，也就是说，所有任务只能在一个线程上完成，一次只能做一件事。Web Worker 的作用，就是为 JavaScript 创造多线程环境，允许主线程创建 Worker 线程，将一些任务分配给后者运行; 那什么时候应该使用Web Worker，看完[这篇文章](https://dassur.ma/things/when-workers/)你或许就有答案了；
